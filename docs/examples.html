BootstrapX - popoverx

Inspired by lecar-red's bootstrapx-clickover, adding hover ability which also performs the same functions with clickover.

See Example

Why?

lecar-red's bootstrapx-clickover is a great extension for Click managed popovers. However it doesn't provide the hover managed function. Although the native popover on bootstrap has the ability on hover, it's hard to be integrated with html content returned from server via ajax. Also it will generate popover instance for each trigger element which is not expected in the most cases. That's also lecar-red's bootstrapx-clickover resolved for us:

  • Allow only single popover at any time
  • onShown and onClose call back give us the chance to interact with server
  • Option to close popover when click is else where in body

Popoverx provide more usage than clickover, not only for click managed function but also for hover. Meanwhile it provides the option to ensure your popover won't exceed the bound of browser.

Popoverx Features

  • Ensure popover inside the broswer boundary
  • Hover trigger popover
  • Hover close delay when trigger popover via hovering which give you the chance to move mouse to popover itself.

Examples

Hover

Button will open popoverx even if you mouse over to popover itself.

Hover for popoverx

Code

Enable popoverx via javascript: 

$('#example').popoverx(options)

Ensure visiable

Ensure the popover won't exceed the boundary of browser.
Actually its placement was set as 'left'

Ensure Visiable

Code

$('#ensure-visiable').popoverx({
  fire_on: 'hover',
  ensure_visiable: true,
  placement: 'left'            
});

Options

Name type default description
fire_on string click tigger event, currently only support 'click' and 'hover'
hover_delay_close number 0 Value in milliseconds. Values greater than 0 will keep popoverx open for giving user the change to mouse to popover itself
ensure_visiable boolean true Ensure the popover won't exceed the boundary of browser.
